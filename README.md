# Instructions

Clone the repository:

    git clone ssh://git@code.seantis.ch:2222/lehr/schnuppern-webanwendung.git

Prepare the environment:

    ./install.sh

Create the database:

    make grades.db

Run the application:

    ./run.sh

# Dependencies

- Python 3
- Virtualenv
- Make
- SQLite
- Pandoc (optional for document preparation)
